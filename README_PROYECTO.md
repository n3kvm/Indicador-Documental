# Indicador Documental

Proyecto copiado limpio para control operativo.

Incluye:
- Ejecutables locales en outputs
- Scripts de lectura/procesamiento en work
- Alternativa GitHub Actions en .github y github_actions
- Azure Function en azure_function
- Paquete limpio de subida a GitHub en PAQUETE_GITHUB_INDICADOR_DOCUMENTAL

No incluye PDFs, caches, __pycache__, resultados Excel/HTML generados ni descargas temporales.
